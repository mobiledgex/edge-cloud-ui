# World map dashboard

### Run 
```
$ npm install
$ npm start
```

### Style Guid

| page Name | component  | css           | 설명                         |
|-----------|------------|---------------|------------------------------|
| main      | main       | main.css      | 메인 전체(칼라, 레이아웃 등) |
|           | worldMap   | worldMap.css  | 지도의 칼라                  |
|           | line Chart | lineChart.css | 라인차트의 스타일            |
